# Geo Calculator

Multi function Geo Location calculator

## Description
This project has the intention of providing a way to make all sorts of Geo calculations.

At the moment contains an implementation of:

1.  Random Western Point contained in one defined square difference.
2.  Random Eastern Point contained in one defined square difference.
3.  Calculates the new coordinate based on a source point, and a latitude/longitude delta.
